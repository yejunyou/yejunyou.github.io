汇编基础-1

Posted on 2019-11-19

汇编的知识点比较杂且多，为方便梳理，整理成思维导图：

语言发展

  • 机器语言
  • 汇编语言
  • 高级语言
    • 编译与反编译的关系
    • 特点
  • 直接访问、控制硬件
  • 和机器指令一一对应
  • 不区分大小写

汇编分类

  • 8086
  • AT&T（Mac、iOS模拟器）
  • ARM（切入式、iOS设备）
  • Win32/Win64
  • …

总线

  • 地址总线 AB：决定CPU的寻址能力
  • 数据总线 DB：决定CPU单次数据传送速度
  • 控制总线CB：决定对其他意见的控制种数

内存

  • 内存单元都有唯一的物理地址
  • 8086的内存大小：
    • 2^20 = 1M (0x00000~0xFFFFF)
    • 0x00000~0x9FFFF 主存地址空间（RAM）
    • 0xAFFFF~0xBFFFF 显存地址空间
    • 0xCFFFF~0xFFFFF 各种ROM地址空间（存储各种硬件/系统信息）

寻址方式

  • 8086有20位地址总线，但是16位结构的CPU
  • 内部用2个16位地址合成1个20位的物理地址 = 段地址 x 16 + 偏移地址
  • 内存的分段管理
    • 目的：便于开发管理
    • 偏移地址最大16位，长度是64KB
    • 数据存在21F60H的2种描述方式：
      • 数据存在内存2000：11F60单元
      • 数据存在内存2000H段中的1F60H单元中

寄存器

  • CPU典型构成
    • 寄存器：信息储存
    • 运算器：信息处理
    • 控制器：控制其他器件工作
    • 8086有14个寄存器
    • 通用寄存器
      • 数据寄存器
        • AX：AH + AL
        • BX：BH + BL
        • CX：CH + CL
        • DX：DH + DL
      • 指针寄存器
        • 堆栈指针 SP
        • 基数指针 BP
      • 变址寄存器
        • 源变址 SI
        • 目的变址 DI
      • 控制寄存器
        • 指令指针 IP
        • 状态指针 FLAG
      • 段寄存器
        • 代码段 CS
        • 数据段 DS
        • 堆栈段 SS
        • 附加段 ES
    • 都是16位，可以存放2个字节
    • 字与字节
      • 字节：byte，1byte = 8bit
      • 字：word，1word = 2byte
    • 段寄存器
      • CS：Code Segment
      • DS：Data Segment
      • SS：Stack Segment
      • ES：Extra Segment

指令

  • CS和IP

    • CS为代码段寄存器，IP为指令指针寄存器，它们指示了CPU当前要读取指令的地址
    • 任意时刻，8086CPU都会将CS:IP指向的指令作为下一条需要取出执行的指令
    • CPU根据什么区分指令和数据

  • jmp指令

    • 统称转义指令
    • 格式:
      • jmp 段地址：偏移地址（jmp CS：IP）
      • mov IP，ax
      • jmp 0x00H

  • DS和[address]

    • 读写内存单元时，写给内存单元的地址
    • 8086不支持直接将数据送入段寄存器（DS）

  • mov指令

    • mov 寄存器，数据
    • mov 寄存器，寄存器
    • mov 寄存器，内存单元
    • mov 内存单元，寄存器
    • mov 段寄存器，寄存器
    • mov 寄存器，段寄存器
    • 不支持 mov 内存单元，内存单元

  • add和sub

栈

- 存储空间，LIFO
- SS作为栈段的段地址，任意时刻，SS：SP指向栈顶元素
- 8086提供了POP和PUSH操作，8086中push和pop操作都是2个字节
- push ax操作过程：
  - sp = sp - 2
  - 把ax中的内容送入ss：sp指向的内存单元
- pop ax操作过程
  - 把ss：sp指向的内存单元送入到ax
  - sp = sp+2
- ex：
  - 交换2个元素
  - 逆序复制

完整的汇编程序

  • 汇编语言的组成
    • 汇编指令：有对应的机器指令，被CPU执行
    • 伪指令：无对应的机器指令，不被CPU执行
  • 注释以分号；开头
